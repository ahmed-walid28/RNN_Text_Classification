# RNN Text Classification

Simple Recurrent Neural Network (RNN) model for text sentiment classification using PyTorch.

## Dataset
Small dataset of positive and negative sentences.

## Model
- Embedding Layer
- RNN Layer
- Fully Connected Layer
- Sigmoid Activation

## Training
- Loss: BCELoss
- Optimizer: Adam
- Epochs: 10

## Run
Open the notebook and run all cells.
